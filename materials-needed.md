# 你需要提供的素材清单

你不用一次性准备完整。先给我“乱的、真实的、你平时真的会用的”内容就可以。

## 第一批最有用的素材

1. 幕布导出的核心课程资料
   - 优先导出 Markdown
   - 如果不方便，Word、PDF、纯文本也可以

2. 你最常讲的 3-5 个案例
   - 可以先匿名化
   - 不用写得很工整
   - 只要包含：问题背景、抽到的牌、你当时怎么解、后来有没有反馈

3. 你的课程/服务介绍
   - 一对一教学怎么卖
   - 团课怎么卖
   - 学生学完能得到什么
   - 你不想吸引哪类学生也可以写

4. 你的个人品牌表达
   - 你希望别人怎么理解你
   - 你和其他塔罗老师最不一样的地方
   - 你的语气更偏温柔、犀利、理性、疗愈、陪伴，还是混合

5. 权限判断
   - 哪些内容可以公开
   - 哪些只给团课
   - 哪些只给一对一
   - 哪些只能你自己看

## 幕布导出建议

如果幕布里有多个主题，建议先按下面方式导出：

- 牌义体系
- 课程主线
- 课堂例子
- 案例复盘
- 学生常见问题
- 私人灵感/碎片笔记

如果现在分不出来，也没关系。你可以先给我一整份，我来帮你拆。

## 案例最小格式

你可以直接按这个格式丢给我：

```text
案例主题：
是否可给学生看：
问题背景：
抽到的牌：
我的解读：
咨询反馈：
这个案例适合教学什么：
需要隐藏的信息：
```

## 第一版权限建议

- 公开访客：品牌介绍、公开文章、少量入门案例
- 团课学生：课件、作业、阶段练习、精选案例
- 一对一学生：完整核心经验、案例复盘、深入讲法
- 仅自己：真实咨询记录、敏感细节、未整理的判断逻辑

第一版可以先用“不同入口/密码页”做轻权限。等内容稳定以后，再升级到账号登录和学生管理。
